# Модель оценки рыночного риска портфеля

VaR/ES, бэктестинг и тесты
покрытия для портфеля из 5 ОФЗ, 10 акций и валютной позиции (USD/EUR).

Полная воспроизводимость обеспечена фиксированным `SEED = 42` (см. `config.py`).

---

## 1. Как запустить

```bash
cd src
pip install pandas numpy scipy statsmodels openpyxl matplotlib scikit-learn arch
python main.py

Все результаты (таблицы CSV и графики PNG) сохраняются в каталог `output/`.
Для интерактивной работы с промежуточными результатами используйте
`risk_model.ipynb` или вызов `main.run_all()` из Python — функция возвращает
словарь со всеми объектами (панель данных, риск-факторы, модели, отчёты).

## 4. Краткое описание входов / преобразований / выходов

- **Входы:** дневные котировки 2021-01-04 … 2025-12-30 (1271 общий торговый день).
- **Преобразования:** очистка → общий календарь → риск-факторы (изменения ставок,
  лог-доходности акций/валют) → PCA кривой → MLE-маргиналы + копула →
  Монте-Карло переоценка → меры риска → бэктест → тесты.
- **Выходы:** таблицы `output/p{2..7}_*.csv`, бонус, графики `output/fig_*.png`.

## 5. Методология и критическое обсуждение (для защиты)

### П.2 — Риск-факторы и PCA
**Выбор:** 3 главные компоненты кривой (из 12 сроков) + 10 лог-доходностей акций
+ 2 лог-доходности валют = **15 факторов**.
- Сроки кривой почти коллинеарны, поэтому PCA применяется к их *посуточным
  изменениям*: 3 компоненты объясняют **95.7%** дисперсии и имеют классическую
  интерпретацию «уровень (76%) / наклон (13%) / кривизна (6.5%)». Это и есть
  требуемое п.2a сокращение размерности.
- **Критика:** акции не сворачивались в PCA, т.к. для полной переоценки нужна
  индивидуальная доходность каждой бумаги; альтернатива — рыночный фактор IMOEX +
  идиосинкразия (однофакторная модель) — даёт меньше факторов, но теряет
  специфический риск. Brent, IMOEX, RTS **не входят** в P&L (этих инструментов
  нет в портфеле), поэтому используются как объясняющие ряды, а не факторы переоценки.
- Описательная статистика: все факторы **стационарны** (ADF, p<0.05), **сильно
  не нормальны** (Jarque-Bera, p<0.05) и **тяжёлохвосты** (большой эксцесс) — это
  напрямую мотивирует выбор распределения Стьюдента в п.3.
- **Корректировки данных:** обнаружены и устранены сплиты GMKN (1:100, апр. 2024)
  и PLZL (1:10, мар. 2025); обвал рынка 24.02.2022 сохранён как реальное движение.

### П.3 — Модели динамики и MLE
**Выбор:** маргинал каждого фактора — распределение **Стьюдента** `t(ν, μ, σ)`,
параметры по **MLE**; зависимость — **гауссова копула** (корреляция нормальных
скоров); инновации **i.i.d.**, горизонт — сумма суточных.
- t-распределение по AIC лучше нормального для **всех 15** факторов; оценённые
  `ν ≈ 1.8–3.4` подтверждают тяжёлые хвосты. Для конечной дисперсии и устойчивого
  Монте-Карло введён порог `ν ≥ 3` (`T_DF_FLOOR`).
- **Критика:** i.i.d.-допущение игнорирует кластеризацию волатильности. Реализована
  альтернатива **GARCH(1,1)-t** (`dynamics.fit_garch`); она точнее в спокойные/бурные
  периоды, но усложняет горизонтную агрегацию. Гауссова копула недооценивает
  совместные экстремумы — возможная замена на t-копулу (хвостовая зависимость).

### П.4 — Справедливая стоимость
- **Облигации:** дисконтирование денежных потоков по КБД. Чистая КБД-цена на
  ~3.6% ниже рыночной (ОФЗ торгуются дороже кривой); индивидуальный **z-spread**
  (≈ −1%) калибруется на дату оценки так, что модель = рынок. ОФЗ 26231 неликвидна
  (нет котировок) → оценивается чисто по КБД, что согласовано (её риск-фактор —
  сама кривая).
- **Акции:** цена акции и есть риск-фактор → полная переоценка позиции.
- **Валюта:** стоимость = объём × курс; курс — риск-фактор.

### П.5 — VaR / ES
- **Монте-Карло** с полной переоценкой (`N_SIM = 50 000`); облигации
  переоцениваются векторно через сдвиг кривой, восстановленный из 3 PCA-компонент.
- **Ежедневная ребалансировка** к постоянным номиналам (п.5c): суточный
  `P&L = Σ Nᵢ·rᵢ`, горизонтный — сумма суточных.
- **Результат (02.12.2025):** 1-дн. VaR₉₉ ≈ **5.79 млн** (2.23% портфеля),
  ES₉₇.₅ ≈ 6.41 млн; 10-дн. VaR ≈ **16.75 млн** (6.44%). Риск **доминирует
  валюта** (200 из 350 млн — это валютная позиция), что экономически ожидаемо.

### П.6–7 — Бэктест и тесты
- Прогноз 1-дн. VaR на каждый торговый день 2025 г.; модель переоценивается на
  расширяющемся окне (без заглядывания вперёд), нагрузки PCA зафиксированы
  (стабильны). **Пробои:** портфель 1, облигации 3, акции 2, валюта 2 при
  ожидаемых ≈ 2.5 — близко к норме.
- Тесты (**Kupiec POF 1995**, **Christoffersen 1998** — независимость и условное
  покрытие, **Engle-Manganelli DQ 2004**): по всем подпортфелям p-value > 0.05,
  гипотеза о корректности оценки **не отвергается**.
- **Критика:** малое число пробоев (≈2–3) даёт низкую мощность тестов; тесты
  независимости/DQ чувствительны к кластеризации, которой при i.i.d.-модели почти нет.

### П.8 — Бонус
- **Опционы на фьючерс MIX** — модель **Блэка-76**; подразумеваемая волатильность
  калибруется по наблюдаемым опционам (Call σ≈31%, Put σ≈21% — улыбка). Расхождение
  пут-колл паритета объясняется неликвидностью пута (OI=18). Портфель 1 = 1 Call ITM
  + 1 Put ITM (пут оценён по калиброванной vol, т.к. рыночной котировки нет).
- **Облигации со встроенными опционами** — модель Блэка для опциона на цену
  облигации (форвард из КБД, σ цены из истории). При ставках ~14% ≫ купона все ОФЗ
  торгуются с дисконтом, поэтому: **putable** (оферта-пут у держателя) заметно
  **дороже** обычной (+177 руб., пут глубоко в деньгах), **callable** ≈ обычной
  (колл эмитента бесполезен). 
- **п.8c (встраивание в основной портфель):** добавить опционные инструменты как
  новые позиции с собственной функцией переоценки `value(risk_factors)` (Блэк-76/
  Блэк), а их риск-факторы (цена фьючерса/индекса, волатильность) — в общий вектор
  факторов и копулу; в `risk_measures.simulate_pnl` добавить ветку переоценки
  опционов вдоль траектории. Облигации с офертой — переоценивать как «обычная ±
  встроенный опцион» на каждом шаге Монте-Карло.

## 6. Ограничения
i.i.d.-динамика (нет волатильностной памяти); гауссова копула (слабая хвостовая
зависимость); z-spread зафиксирован на дату оценки; горизонт 10 дней без учёта
старения облигаций (эффект ≪ риска кривой). Все пункты вынесены как направления
улучшения и реализуемы поверх текущей модульной архитектуры.
