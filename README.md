# Модель оценки рыночного риска портфеля

VaR/ES, бэктестинг и тесты
покрытия для портфеля из 5 ОФЗ, 10 акций и валютной позиции (USD/EUR).

Полная воспроизводимость обеспечена фиксированным `SEED = 42`

---

## 1. Как запустить

```bash
cd src
pip install pandas numpy scipy statsmodels openpyxl matplotlib scikit-learn arch
python main.py

Все результаты (таблицы CSV и графики PNG) сохраняются в каталог `output/`.
```

## 2. Структура проекта

| Файл | Пункт | Назначение |
|------|-------|-----------|
| `config.py` | — | Состав портфеля, пути, параметры, `SEED` |
| `data_loader.py` | 1 | Загрузка и очистка данных, общий календарь, расписания купонов |
| `risk_factors.py` | 2 | Риск-факторы, **PCA** кривой, описательная статистика |
| `dynamics.py` | 3 | Маргиналы Стьюдента (**MLE**), гауссова копула, симуляция |
| `pricing.py` | 4 | Справедливая стоимость инструментов, проверка точности |
| `risk_measures.py` | 5 | **Монте-Карло VaR/ES** с ежедневной ребалансировкой |
| `backtesting.py` | 6–7 | Бэктест VaR, тесты Kupiec / Christoffersen / Engle-Manganelli |
| `bonus.py` | 8 | Блэк-76 (опционы на фьючерс), облигации со встроенными опционами |
| `plots.py` | 2c | Все графики |
| `main.py` | все |

## 3. Данные (data provenance)

Все ряды — публичные данные Московской биржи и ЦБ РФ.

| Данные | Файл | Первоисточник | Что означает |
|--------|------|---------------|--------------|
| Кривая бескупонной доходности (12 сроков 0.25–30 лет) | `Процентные ставки.xlsx` | MOEX, методика Nelson-Siegel-Svensson по сделкам с ОФЗ | Безрисковая zero-ставка, % годовых |
| Котировки 5 ОФЗ | `Котировки облигации.csv` | MOEX ISS, режим TQOB | Чистая цена (% номинала), НКД, купон |
| Котировки 10 акций | `Котировки акции.csv` | MOEX ISS, режим TQBR | Цена закрытия, руб. |
| Индексы, курсы USD/EUR | `Индексы и курсы.csv` | MOEX (IMOEX, RTSI, фиксинг валют) | Уровни индексов; руб. за 1 ед. валюты |
| Brent | `нефть-brent.xlsx` | Биржевая цена | Контрольный/объясняющий фактор |
| Фьючерс и опционы MIX | `MIX_Options_2025-10-17.csv` | MOEX FORTS | Цена фьючерса/опционов (бонус) |

## 4. Краткое описание входов / преобразований / выходов

- **Входы:** дневные котировки 2021-01-04 … 2025-12-30 (1271 общий торговый день).
- **Преобразования:** очистка, общий календарь, риск-факторы (изменения ставок,
  лог-доходности акций/валют), PCA кривой, MLE-маргиналы + копула, 
  Монте-Карло переоценка, меры риска, бэктест, тесты.
- **Выходы:** таблицы `output/p{2..7}_*.csv`, бонус, графики `output/fig_*.png`.
